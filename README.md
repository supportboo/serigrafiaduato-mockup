### Qué la define
- **Enfoque industrial B2B** (no B2C creativo): piezas, trazabilidad, envases/cilíndrico, señalética, ropa laboral/EPI, consolidación de proveedores.
- Sistema de diseño **«Acero & Seguridad»**: grafito/acero + acento naranja seguridad; tipografía Inter + JetBrains Mono.
- Grafismo de marca (gota de tinta Duato) estampado por las secciones.
- Imágenes **hiperrealistas generadas** (Nano Banana Pro) + fotos reales del taller.
- FX: hero slider, bento visual, contadores, barrido láser, vídeos de técnica, parallax sobrio. Accesible y `prefers-reduced-motion`.
- Datos verificados; SLA/plazos/certificaciones sin confirmar aparecen como «consultar» (nada inventado).
