# Prompts rápidos · listos para copy-paste

**32 prompts** organizados por orden de prioridad. Cada uno indica qué tool usar y dónde va en el mockup.

**Coste total estimado**: ~22 €

**Reglas globales aplicadas a TODOS**:
- Cero caras humanas reconocibles
- Cero marcas/logos competencia
- Paleta cream `#EFEAE0` + ink `#0A0A0A` + spot Duato `#F37920`
- Mood industrial-warm, no neón/sci-fi
- Grain natural, no AI-slop liso
- Iluminación cálida 3200-3800K

**Negative prompt común** (pega en el campo Negative de Luma siempre):
```
cartoon, anime, 3d render, plastic, neon, futuristic, cgi, glossy, watermark, text overlay, ugly hands, deformed, smiling people, faces visible, logos, brand names
```

## DÓNDE PEGAR LOS PROMPTS

- **Luma Dream Machine**: https://lumalabs.ai/dream-machine
  - Click "Create" → pega el prompt → set Aspect Ratio + Loop ON
  - Default 5s · escoge 8s para hero
  - **Negative**: pega el negativo común en el campo Negative

- **Imagen 4.0** (gratis · Gemini): https://gemini.google.com/
  - "Generate image" → pega prompt → ajusta aspect ratio

- **Higgsfield Seedance 1.5** (opcional para hero scroll-driven v3.1): https://higgsfield.ai
  - Start frame + end frame upload → texto del prompt
  - 12 créd (~4€) por vídeo
